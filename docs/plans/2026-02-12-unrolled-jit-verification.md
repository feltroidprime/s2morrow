# Fully Unrolled JIT Verification

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Eliminate all loops and intermediate arrays from Falcon-512 verification by using `multi_pop_front`, `#[inline(always)]` generated functions returning fixed-size arrays, and JIT validation.

**Architecture:** Input data stays as `felt252` until point of use. Three generated `#[inline(always)]` functions handle: (1) NTT + validation in one unbox, (2) felt252→Zq validation only, (3) unrolled verify+norm. The verify function orchestrates via `multi_pop_front` on felt252 spans. No loops, no intermediate arrays.

**Tech Stack:** Python code generator (`cairo_gen/`), Cairo 2.15, BoundedInt, `multi_pop_front`, fixed-size arrays `[T; 512]`.

**Key design insight:** `#[inline(always)]` functions returning `[Zq; 512]` make their output variables directly accessible in the caller's scope — no copy, no array allocation.

## Task 2: Create the verify code generator

**Files:**

**Step 1: Create `cairo_gen/circuits/verify.py`:**

```python
"""Generate fully unrolled Falcon-512 verification functions.

Generated functions (all #[inline(always)]):
- ntt_and_validate: NTT + input validation in one unbox
- validate_felt252_512: input validation only (for pk_ntt)
- verify_norm_512: unrolled hint check + norm accumulation

Regenerate with:
    python -m cairo_gen.circuits.regenerate verify --n 512
"""


class VerifyGenerator:
    """Generates unrolled verification Cairo code for Falcon-512."""

    def __init__(self, n: int = 512):
        self.n = n

    def _var_list(self, prefix: str) -> str:
        """Generate comma-separated variable list: prefix0, prefix1, ..."""
        return ", ".join(f"{prefix}{i}" for i in range(self.n))

    def generate_full(self) -> str:
        """Generate the complete verify_unrolled.cairo file."""
        parts = [
            self._header(),
            self._imports(),
            "",
            self._ntt_and_validate(),
            "",
            self._validate_only(),
            "",
            self._verify_norm(),
            "",
        ]
        return "\n".join(parts)

    def _header(self) -> str:
        return (
            "// Auto-generated by cairo_gen/circuits/verify.py\n"
            "// DO NOT EDIT MANUALLY - regenerate with: "
            "python -m cairo_gen.circuits.regenerate verify --n 512\n"
        )

    def _imports(self) -> str:
        return "\n".join([
            f"use crate::ntt_felt252::ntt_{self.n}_inner;",
            "use crate::zq::{Zq, mul_mod, sub_mod, center_and_square};",
            "use corelib_imports::bounded_int::downcast;",
        ])

    def _ntt_and_validate(self) -> str:
        n = self.n
        destructure = self._var_list("f")
        validate_lines = "\n    ".join(
            f"let v{i}: Zq = downcast(*f{i}).expect('invalid Zq');"
            for i in range(n)
        )
        ntt_args = ", ".join(f"*f{i}" for i in range(n))
        ntt_rets = self._var_list("r")
        v_arr = self._var_list("v")
        r_arr = self._var_list("r")

        return f"""/// NTT + validation in one unbox. Returns (validated_input, ntt_output).
#[inline(always)]
pub fn ntt_and_validate(input: @Box<[felt252; {n}]>) -> ([Zq; {n}], [Zq; {n}]) {{
    let [{destructure}] = input.unbox();
    {validate_lines}
    let ({ntt_rets}) = ntt_{n}_inner({ntt_args});
    ([{v_arr}], [{r_arr}])
}}"""

    def _validate_only(self) -> str:
        n = self.n
        destructure = self._var_list("f")
        validate_lines = "\n    ".join(
            f"let v{i}: Zq = downcast(*f{i}).expect('invalid Zq');"
            for i in range(n)
        )
        v_arr = self._var_list("v")

        return f"""/// Validate 512 felt252 values as Zq (no NTT).
#[inline(always)]
pub fn validate_felt252_512(input: @Box<[felt252; {n}]>) -> [Zq; {n}] {{
    let [{destructure}] = input.unbox();
    {validate_lines}
    [{v_arr}]
}}"""

    def _verify_norm(self) -> str:
        n = self.n
        s1n = self._var_list("s1n")
        hn = self._var_list("hn")
        pk = self._var_list("pk")
        msg = self._var_list("msg")
        h = self._var_list("h")
        sv = self._var_list("sv")

        lines = []
        for i in range(n):
            lines.append(
                f"    assert(mul_mod(s1n{i}, pk{i}) == hn{i}, 'hint mismatch');"
            )
            if i == 0:
                lines.append(
                    f"    let acc = center_and_square(sub_mod(*msg{i}, h{i}))"
                    f" + center_and_square(sv{i});"
                )
            else:
                lines.append(
                    f"    let acc = acc + center_and_square(sub_mod(*msg{i}, h{i}))"
                    f" + center_and_square(sv{i});"
                )
        body = "\n".join(lines)

        return f"""/// Unrolled verify + norm: checks hint correctness and computes ||s0||^2 + ||s1||^2.
#[inline(always)]
pub fn verify_norm_512(
    s1_ntt: [Zq; {n}],
    hint_ntt: [Zq; {n}],
    pk_zq: [Zq; {n}],
    msg_point: @Box<[Zq; {n}]>,
    hint_zq: [Zq; {n}],
    s1_zq: [Zq; {n}],
) -> felt252 {{
    let [{s1n}] = s1_ntt;
    let [{hn}] = hint_ntt;
    let [{pk}] = pk_zq;
    let [{msg}] = (*msg_point.unbox());
    let [{h}] = hint_zq;
    let [{sv}] = s1_zq;
{body}
    acc
}}"""
```

```cairo
// SPDX-FileCopyrightText: 2025 StarkWare Industries Ltd.

// SPDX-License-Identifier: MIT

use crate::verify_unrolled::{ntt_and_validate, validate_felt252_512, verify_norm_512};
use crate::zq::Zq;
use falcon::types::{FalconPublicKey, FalconSignatureWithHint, HashToPoint};

/// Signature bound for Falcon-512
const SIG_BOUND_512: u64 = 34034726;

/// Verify a Falcon signature using the hint-based approach.
/// Computes msg_point internally via hash_to_point.
/// Cost: 2 NTTs, 0 INTTs.
pub fn verify<H, +HashToPoint<H>, +Drop<H>>(
    pk: @FalconPublicKey, sig_with_hint: FalconSignatureWithHint, message: Span<felt252>,
) -> bool {
    let msg_point = HashToPoint::<H>::hash_to_point(message, sig_with_hint.signature.salt.span());
    verify_with_msg_point(pk, sig_with_hint, msg_point.span())
}

/// Verify with a pre-computed msg_point (useful for testing without hash_to_point).
///
/// Fully unrolled: 2 NTTs + 1 fused pass for hint verification + norm computation.
/// Zero loops, zero intermediate arrays.
pub fn verify_with_msg_point(
    pk: @FalconPublicKey,
    sig_with_hint: FalconSignatureWithHint,
    msg_point: Span<Zq>,
) -> bool {
    let mut s1_span = sig_with_hint.signature.s1.span();
    let mut pk_span = pk.h_ntt.span();
    let mut hint_span = sig_with_hint.hint.mul_hint.span();
    let mut msg_span = msg_point;

    // Bulk-read all inputs via multi_pop_front (4 cheap libfunc calls)
    let s1_box = s1_span.multi_pop_front::<512>().expect('s1 must be 512');
    let pk_box = pk_span.multi_pop_front::<512>().expect('pk must be 512');
    let hint_box = hint_span.multi_pop_front::<512>().expect('hint must be 512');
    let msg_box = msg_span.multi_pop_front::<512>().expect('msg must be 512');

    // NTT + validate s1 and mul_hint (#[inline(always)] → vars in scope)
    let (s1_zq, s1_ntt) = ntt_and_validate(s1_box);
    let (hint_zq, hint_ntt) = ntt_and_validate(hint_box);

    // Validate pk_ntt only (already in NTT domain, no NTT needed)
    let pk_zq = validate_felt252_512(pk_box);

    // Unrolled verify + norm — zero loops, zero arrays
    let norm = verify_norm_512(s1_ntt, hint_ntt, pk_zq, msg_box, hint_zq, s1_zq);

    let norm_u64: u64 = norm.try_into().unwrap();
    norm_u64 <= SIG_BOUND_512
}
```

## Risk: Compiler limits

The generated `verify_unrolled.cairo` will be ~2000+ lines (before formatting). With two inlined NTTs (~10K lines each via `ntt_512_inner`) and the unrolled verify loop, the effective inlined code in `verify_with_msg_point` could be ~25K lines. The Cairo compiler (v2.15) may hit memory or time limits.

**Mitigation:** If compilation fails or takes too long, remove `#[inline(always)]` from `verify_norm_512` (keep the loop version for that part) and only inline `ntt_and_validate` + `validate_felt252_512`. This still eliminates the NTT output arrays and upcast overhead.

**Mitigation 2:** If `ntt_and_validate` is too large to inline, split it: keep `ntt_512_inner` as a regular function call (it already is), and only inline the validation + destructuring wrapper.

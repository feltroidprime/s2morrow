## Key Architecture Rules for Implementation
1. Landing sections = React Server Components (zero client JS)
2. Interactive sections = client components via `next/dynamic` with `ssr: false`
3. All crypto runs client-side only via WASM
4. Secret keys never leave the browser
5. State via Effect Atoms (@effect-atom/atom, @effect-atom/atom-react)
6. Tailwind CSS v4 with CSS-first config (no tailwind.config.js)
